# Java Eat - Composant "Plats et Utilisateurs"


Ce dépôt contient l'API REST gérant les plats et les utilisateurs pour le projet Java Eat (Architecture Logicielle R4.01). 


## Technologies utilisées

* Jakarta EE / Java 21

* API REST (JAX-RS)

* Serveur d'application : GlassFish 7.0.25

* Gestionnaire de dépendances : Maven
